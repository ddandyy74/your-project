# inputs/ - 外部资料目录

> **备注**: 放外部提供/收集的原始资料，不改动原文

## 目录结构

```
inputs/
├── brief/          # 原始需求、想法、业务说明（你写的）
├── references/     # 参考资料（论文/链接摘录/竞品资料）
├── screenshots/    # 截图
└── attachments/    # pdf、docx、表格等附件（原始件）
```

## 使用规则

- ✅ 放入这里的文件**不修改原文**
- ✅ 原始资料存档，方便追溯
- ✅ 引用时注明来源
- ❌ 不要直接把原始内容复制到 docs/
