## 测试策略

### 测试金字塔

```
        /\
       /  \     E2E 测试 (少)
      /____\
     /      \   集成测试 (中)
    /________\
   /          \  单元测试 (多)
  /____________\
```

### 测试覆盖率要求

| 类型 | 覆盖率要求 |
|------|------------|
| 单元测试 | >= 80% |
| 集成测试 | >= 60% |
| E2E 测试 | 核心流程 |

## 测试环境

| 环境 | 用途 | 数据 |
|------|------|------|
| Dev | 开发测试 | 模拟数据 |
| Staging | 预发布测试 | 脱敏生产数据 |
| Production | 正式环境 | 真实数据 |
