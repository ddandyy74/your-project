# 03_acceptance.md - 验收标准

> 可测试、可判定的验收条件

## 验收原则

- 每个需求必须有明确的验收标准
- 验收标准必须是可测试的
- 优先级 P0 必须 100% 通过

## 性能验收标准

| 指标 | 目标值 | 测试方法 |
|------|--------|----------|
| 响应时间 | < 200ms | 性能测试 |
| 并发用户 | >= 100 | 压力测试 |
